tabularSelect

tabularSelect is a jQuery UI widget that represents multiple select boxes in a more user-friendly control for selecting and deselecting options. The plugin is written from the ground up but is derived from and designed to replace the ui.multiSelect widget written by Michael Aufreiter and Yanick Rochon, with an aim of making it faster and more customizable. A large portion of code has been adapted from the ui.multiSelect project as well as some ideas and techniques from Chosen, by Patrick Filler. For more information, check out the online documentation.

Theming

tabularSelect utilizes the jQuery UI Themeroller and can be themed and modified via CSS. To change the theme, select from one of the default themes from the "Switch theme" drop down.

Use

tabularSelect requires the following:

  • jQuery (minimum version to be determined)
  • jQuery UI (minimum version to be determined)
  • Download ca.tabularSelect.js and ca.tabularSelect.css
  • call .tabularSelect() on select object
